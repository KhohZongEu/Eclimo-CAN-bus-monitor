# Eclimo CAN bus monitor

> Change WiFi name & password + Add database URL
> ```
> #define WIFI_SSID "YOUR_WIFI_NAME"
> #define WIFI_PASSWORD "YOUR_WIFI_PASSWORD"
>
> #define DATABASE_URL "YOUR_DATABASE_URL"
> ```
